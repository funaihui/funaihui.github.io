树莓派折腾记——安装系统

发表于2020-12-21|更新于2020-12-21|树莓派
|阅读量:

前言：说下为什么会入手树莓派呢！记得是一个同事给我说他用树莓派搭建了一个私有云盘，安全、下载速度快、自己买硬盘想要多大空间就买多大空间的。还能学习到一些知识，就这样我被种草了，就开始了折腾树莓派。

什么是树莓派

&emsp;&emsp;在开始系统安装之前，先了解一下什么是树莓派。先看下树莓派的图片

树莓派（英语：Raspberry Pi）是基于Linux的单片机电脑，由英国树莓派基金会开发，目的是以低价硬件及自由软件促进学校的基本计算机科学教育。

树莓派每一代均使用博通（Broadcom）出产的ARM架构处理器，如今生产的机型内存在2GB和8GB之间，主要使用SD卡或者TF卡作为存储媒体，配备USB接口、HDMI的视频输出（支持声音输出）和RCA端子输出，内置Ethernet/WLAN/Bluetooth网络链接的方式（依据型号决定），并且可使用多种操作系统。产品线型号分为A型、B型、Zero型和ComputeModule计算卡。

以上内容来自维基百科，简单的说树莓派就是一个微型电脑。

选择操作系统

&emsp;&emsp;树莓派有一下操作系统可供选择：

  • Raspberry Pi OS（原为Raspbian）是为树莓派基于Debian开发的操作系统。从2015年起，树莓派基金会正式将其作为树莓派的官方操作系统。
  • **OSMC**（开源媒体中心）是一款免费，简单，开源且易于使用的独立 Kodi 操作系统，几乎可以播放任何媒体格式。
  • Ubuntu Core 是为物联网应用程序设计的 Ubuntu 版本。 Ubuntu 是世界上最受欢迎的基于 Linux 的操作系统，拥有 20 多种衍生产品，并且它拥有一个活跃且热情的论坛，在 Raspberry Pi 上使用 Ubuntu Snappy Core 很容易启动和运行。

这里选择三个做个简单的介绍，其实支持树莓派的系统有很多，这里就不一一进行介绍了，想要了解更多系统支持树莓派的系统，请移步到这里。

这里我推荐一款由OPENFANS开源社区 & 树莓派爱好者社区 联合出品的 Debian-Pi-Aarch64系统，关于这个系统的介绍，可以到官网进行详细的了解（文末我会放出这个系统的官网）。下文介绍的安装系统，也是以这个系统来介绍。

安装操作系统

需要的硬件及软件

  • 32G以上的SD卡一个（官方推荐 使用 A1 级别的 SD 卡），怎么辨别是否是A1级的SD卡可以看下图

  • 读卡器一个（用来写系统到SD卡中）

  • Debian-Pi-Aarch64系统

  • 树莓派

  • Etcher用来将系统写入SD卡的软件，这个软件Mac和Windows都支持。

将操作系统镜像写入SD卡

  1. 打开上面的Etcher软件，界面如下

  2. 点击Select image选择我们下载好的镜像，选择之后会让我们选择要写入的SD卡，界面如下。

    选择好SD卡，等待刷入成功即可。

  3. 将SD卡插入树莓派的SD卡插槽内，然后树莓派开机。

验证系统是否安装成功

Debian-Pi-Aarch64系统既有有桌面的操作系统，也有无桌面的操作系统，如果你安装是带界面的系统，则在开机时连接显示器即可知道系统安装的进度和结果。如果你安装的是无桌面的操作系统，要验证系统是否安装成功，需要以下步骤：

  1. 查看路由器，查找路由器为树莓派分配的内网ip，

    查看客户端名称为raspbian的内网IP，我这里是192.168.1.5。

  2. 用连接同一网络的电脑，打开浏览器，在浏览器输入xxx.xxx.xxx.xxx:9090，’xxx.xxx.xxx.xxx’是你树莓派的内网ip，这里我的是192.168.1.5，则在浏览器输入的是192.168.1.5:9090，然后回车，如果能看到下面的界面

    则说明安装成功。

    注意:如果你回车后看到的是下面的界面

    这是浏览器提示你访问的地址不是https的，为不安全的网址，这个不用管，点击下面的按钮，继续访问就OK了。

到这里，树莓派的系统就安装完成了。
